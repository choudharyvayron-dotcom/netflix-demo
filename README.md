I built a Netflix clone from scratch using HTML and CSS as part of my web development learning journey.
The project includes a Netflix-inspired homepage featuring a hero section, trending movie cards, feature sections with custom SVG icons,
language selectors, email signup forms, and a responsive footer. While developing this project, I learned how to structure webpages using semantic HTML,
style layouts with CSS, create and customize SVG icons, use Flexbox and Grid for positioning, and manage my code using Git and GitHub. 
This project reflects my problem-solving skills, attention to detail, and enthusiasm for continuously improving my front-end development abilities.
